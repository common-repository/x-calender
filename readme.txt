== Description ==

### Add an events calendar to your site real fast, real easy!

**NEX-Events Calendar** is a high quality and simple to use drag and drop events calendar. Drag over 450 font awesome retina-ready vector icons right onto you calendar. Repeat events daily, monthly or yearly with exclusion and inclusion dates with lots of variations.

If you have a need to add events that comes with a customizable good looking calendar display, you came to the right place!! 

**With NEX-Events Calendar you can notify your visitors of events in not time!**

= Front/Backend Calendar view =
Events are listed in a full calendar view on your website with easy browsing by year, month or day. The admin interface gives you exactly the same calendar view to add you events.

= 25 different preset color schemes to choose from =
Themes/Color Schemes are applied accordingly to your calendar and event popups

= Responsive design =
With liquid layout design, NEX-Events Calendar will work on all devices...Iphone, Ipad, Tablet, Desktop, Flat screen TV, etc...

= Add recurring events =
This is a very awesome feature that comes with quite a number of combinations and execptions for event repititions. Programme your recurring events as follows:
* Daily: Repeats the event daily. - Choose days to exclude for example: exclude Saterday and Sunday to exclude weekends on the daily event repitition 
* Weekly: Repeats the event on any selected week days. 
* Monthly: Repeats the event monthly.- Choose day of the month and/or; - Choose the 1st and/or 2nd and/or 3rd and/or 4th combination of selected weekdays to repeat monthly.
* Repeat any recurring event to a specified end date 
* Choose what specific dates should be excluded in the repitition 

= Google maps integration =
Simple add your event location (venue name or address) and include google maps for direction to your events

= What you see is what you get (WYSIWYG) =
What you see on your backend is what your visitors will see on your website!

== Installation ==

1. After your download log into your WordPress admin interface
2. After login click on  Plugins from the left  hand menu
3. Click on “Add new”
4. Click on “Upload”
5. Browse to the directory you downloaded the plugin to and click “Install Now”
6. After Wordpress has finished unpacking the file click on “Activate Plugin”
7. After the plugin has been activated you will notice a new menu item on the left hand navigation labelled X Calendar

== Frequently Asked Questions ==

= Why is the user-interface not displaying correctly? =
Your theme's style.css might be interfering with the default css of the plugin. To customize the way the plugin is displayed in your theme either create styles overwriting the the styles called from X Calendar/css/ui.css or customize X Calendar/css/ui.css to suit your theme. 
This plugin was developed following WP best practice and will work best with themes thats developed using the same standards.

= How do I make suggestions or report bugs for this plugin? =
Email questions,bugs and suggestions to pluginsexpress@basixonline.net
